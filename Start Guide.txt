1) Clone from Github
2) Create a file '.env' in the server folder and paste following content:

	PORT=5000
	JWT_TOKEN=SECRET
	PG_HOST=localhost
	PG_USER=admin
	PG_PORT=5432
	PG_PASSWORD=admin
	PG_DATABASE=thebloggr

3) Go inside the server folder and type the command 'npm i'
4) Go inside the client folder and type the command 'npm i --legacy-peer-deps'
5) Inside the client folder type command 'npm start' to start the frontend
6) Inside the server folder type command 'npx nodemon server' to start the server
